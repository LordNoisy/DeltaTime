# DeltaTime Minecraft Mod

This mod aims to separate certain functions from TPS, instead opting for them to use a delta time approach.

## Features

FINISHED:

PLANNED:
- Separate Eating
- Separate XP Pickup
- Separate Item Pickup
- More as I think of them :)
